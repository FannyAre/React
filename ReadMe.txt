******Página Web de Recetas de Comida******

Esta página web fue desarrollada con React y se conecta a la API de TheMealDB para mostrar platos con sus respectivas recetas y categorías.

Funcionalidades
 - Lista de platos con recetas. 
 - Filtro de platos por ingrediente o categoría.
 - Selección y deselección de platos favoritos.
 - Sección de favoritos para ver los platos marcados como tal.
